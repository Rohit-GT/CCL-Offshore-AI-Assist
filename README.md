# CCL Offshore AI Assist

AI-powered Roster and Schedule Assistant for Offshore Teams.

## Features
- **Schedule Querying:** Ask questions about shift schedules, rosters, and team assignments using natural language.
- **RAG Architecture:** Powered by Gemini AI, SentenceTransformers, and ChromaDB vector search.
- **Unified Interface:** Integrated FastAPI backend and static web dashboard.
